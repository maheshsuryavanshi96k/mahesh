#  AWS EC2 Storage Expansion Project

##  About
This project demonstrates how to increase and manage EC2 storage in AWS for better performance and scalability.

---

##  Services Used
- Amazon EC2
- Amazon EBS
- AWS Management Console
- Linux

---

##  Tasks Performed
- Created and attached EBS volume
- Extended EC2 storage size
- Verified disk attachment
- Mounted storage on Linux server
